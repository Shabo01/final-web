React Student Catalog

Run locally:

1. Open terminal in `react-app` folder:

```powershell
cd "C:\Users\HP\OneDrive\Рабочий стол\final-web\react-app"
npm install
npm run dev
```

2. Open the URL printed by Vite (usually http://localhost:5173).

Notes:
- Uses `react-router-dom` with routes `/` and `/students/:id`.
- Bootstrap classes used for layout and spacing.
- Student data in `src/data/students.js`.
